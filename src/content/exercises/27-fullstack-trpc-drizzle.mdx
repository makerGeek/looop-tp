## Goal
Read a minimal full-stack slice: Drizzle schema → tRPC router → React client
that uses TanStack Query via `@trpc/tanstack-react-query`.

## Concepts
End-to-end types are the whole point: your DB types flow into your router,
into your client, into your components. No codegen step, no drift.

## Read-along
1. `db/schema.ts` — Drizzle table definitions (SQLite).
2. `server/routers/todos.ts` — a tRPC router with `list` / `add` procedures.
3. `components/TodoList.tsx` — the client using `useQuery` + `useMutation`.

## Common pitfalls
- Sharing server-only code to the client — use `"server-only"` or split packages.
