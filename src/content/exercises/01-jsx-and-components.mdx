## Goal
Render a small greeting card component that takes a `name` prop and shows it
inside an `<h2>`.

## Concepts
In React 19, components are still just functions. JSX looks like HTML but is
really `React.createElement` under the hood — which is why attribute names use
camelCase (`className`, `onClick`) and why you need a single root (or a fragment).

## Steps
1. Define a `Greeting` component that accepts a `name: string` prop.
2. Return a `<section>` containing an `<h2>` that says `Hello, {name}!`.
3. Render `<Greeting name="Dana" />` from `App`.

## Common pitfalls
- Using `class=` instead of `className=`.
- Forgetting the capital letter — lowercase components are treated as HTML tags.

## Stretch
Add a second prop `emoji?: string` that renders before the name when provided.
