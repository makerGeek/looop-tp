## Goal
Build a counter with increment, decrement, and reset buttons.

## Concepts
`useState` returns `[value, setter]`. React re-renders whenever the setter is
called with a new reference. Updates are queued and batched — use the updater
function to guarantee correct sequencing when state depends on the previous value.

## Steps
1. Import `useState` from `react`.
2. Create `[count, setCount]` initialized to `0`.
3. Wire three buttons: `+`, `-`, and `reset`.
4. Display the count in a big heading.

## Common pitfalls
- Mutating the value (`count++`) — React only notices new values.
- Calling `setCount` in a render without a handler — that causes an infinite loop.

## Stretch
Add a `step: number` state that controls how much `+`/`-` change at a time.
