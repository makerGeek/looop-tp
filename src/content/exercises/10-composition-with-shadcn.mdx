## Goal
Build a settings card that opens a confirmation dialog when the user clicks
"Delete account".

## Concepts
shadcn/ui is a pattern, not a library — you copy primitives into your repo and
compose them. For this sandbox we use inline minimal primitives to show the
composition model without needing to install Radix.

## Steps
1. Create a `Card` with header, body, and a `Delete` button.
2. When clicked, open a `Dialog` asking for confirmation.
3. The dialog has `Cancel` and `Confirm` actions.

## Common pitfalls
- Putting everything in one component — composition makes swapping parts trivial.

## Stretch
Add an `onConfirm` prop to the dialog so the caller decides what to do.
