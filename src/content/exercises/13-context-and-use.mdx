## Goal
Share a theme value through context and read it deep in the tree with `use()`.

## Concepts
`useContext` still works in React 19, but `use()` is the successor — it unifies
the API for reading contexts and awaiting promises. It's valid inside
conditionals and loops.

## Steps
1. Create `const ThemeContext = createContext<'light' | 'dark'>('light')`.
2. Wrap the app in `<ThemeContext.Provider value="dark">`.
3. Read it deep down with `const theme = use(ThemeContext)`.

## Common pitfalls
- Passing new object literals as `value` every render — memoize them if identity matters.

## Stretch
Also expose a `setTheme` function by putting `{ theme, setTheme }` in the context value.
