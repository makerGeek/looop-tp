## Goal
Build a live clock that ticks every second, and cleans up its interval on unmount.

## Concepts
`useEffect` synchronizes React state with the outside world (timers, subscriptions,
direct DOM calls). The cleanup function runs before the next effect and on unmount.

## Steps
1. Store a `Date` in state, initialized with `new Date()`.
2. In a `useEffect`, call `setInterval(() => setNow(new Date()), 1000)`.
3. Return `() => clearInterval(id)` from the effect.
4. Display `now.toLocaleTimeString()`.

## Common pitfalls
- Forgetting the cleanup — you'll leak timers every remount.
- Missing deps warning — if you read a prop inside, list it in the array.

## Stretch
Add a pause button that stops the interval via state.
