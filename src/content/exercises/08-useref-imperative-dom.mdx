## Goal
Add a "focus input" button that imperatively calls `.focus()` on an input and a
"measure box" button that reads the box's height.

## Concepts
Refs are escape hatches to the DOM. `useRef` returns a stable object whose
`.current` is mutable without causing re-renders. In React 19, function
components can accept a `ref` prop directly — no more `forwardRef` boilerplate.

## Steps
1. Create `inputRef` and attach it to the `<input>`.
2. Create `boxRef` and attach it to a `<div>`.
3. Wire two buttons: one calls `inputRef.current?.focus()`, the other reads `boxRef.current?.getBoundingClientRect()`.

## Common pitfalls
- Accessing `.current` during render — refs aren't populated until after mount.

## Stretch
Display the measured box dimensions next to the button.
