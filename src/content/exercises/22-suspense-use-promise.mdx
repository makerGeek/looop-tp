## Goal
Render a component that `use()`s a promise, wrapped in a `<Suspense>` fallback.

## Concepts
React 19's `use()` hook can unwrap promises (and contexts). When called on an
unresolved promise, the nearest `<Suspense>` boundary shows its fallback until
the promise resolves. Great for progressive rendering.

## Steps
1. Build a `fetchProfile()` that resolves after 1s.
2. Create the promise once at the top (outside the suspending component).
3. Inside the component, `const data = use(promise)` and render it.
4. Wrap the component in `<Suspense fallback={<Skeleton />}>`.

## Common pitfalls
- Creating the promise inside the suspending component — it re-suspends forever.

## Stretch
Add a `refresh` button that creates a new promise and forces re-suspension.
