## Goal
Build a cart store with two slices (items + discount) and selectors.

## Concepts
Zustand is a tiny flux-ish store that reads like plain state + functions.
Slices keep domains separated; selectors keep components from re-rendering on
unrelated changes.

## Steps
1. Define `itemsSlice` (add, remove, clear) and `discountSlice` (setCode).
2. Compose them in a single `create(...)` call.
3. Subscribe only to what each component needs.

## Common pitfalls
- Subscribing to the whole state — components re-render on every change.

## Stretch
Add a derived selector for `total` that multiplies items and applies the discount.
